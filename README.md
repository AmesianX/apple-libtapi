# Apple TAPI library, Version 1.30.0 # 

Sources taken from: 

https://opensource.apple.com/tarballs/clang  
https://opensource.apple.com/tarballs/tapi 

## Dependencies: ##

CMake, Python and GCC 4.8+ or Clang

## Installation: ##

    [INSTALLPREFIX=<prefix>] ./build.sh  
    ./install.sh
