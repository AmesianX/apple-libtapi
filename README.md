# Apple TAPI library, Version 2.0.0 # 

Sources taken from: 

https://github.com/ributzka/tapi  
https://opensource.apple.com/tarballs/tapi  

## Dependencies: ##

CMake, Python and GCC 4.8+ or Clang

## Installation: ##

    [INSTALLPREFIX=<prefix>] ./build.sh  
    ./install.sh
